# Csharphello

A beginner-friendly C# Console Application created as part of a learning capstone project.

---

##  Project Overview
This project demonstrates a minimal working C# application using the .NET SDK.  
It prints a simple message to the console.

---

##  Requirements
- .NET SDK 8+
- Windows, Linux, or macOS
- VS Code or any preferred editor

```bash
dotnet --version
```

## 2. Quick Summary of the Technology

C# is a modern, object-oriented programming language developed by Microsoft for building scalable and robust applications. It runs on the .NET platform and supports strong type safety, garbage collection, and cross-platform development.

**Where is it used?**  
- Desktop applications (Windows Forms, WPF)  
- Web APIs and backends (ASP.NET)
